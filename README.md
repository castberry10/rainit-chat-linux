## 시작 방법

[terminal]  
git clone https://github.com/castberry10/rainit-chat-linux   
cd rainit-chat-linux  
chmod +x install_rainit.sh uninstall_rainit.sh
  
### 설치  
sudo ./install_rainit.sh   
  
### 실행  
rainit  
  
### 삭제  
sudo ./uninstall_rainit.sh
